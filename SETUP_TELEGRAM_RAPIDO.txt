PASO 1: OBTENER TOKEN DE TELEGRAM (3 minutos)
════════════════════════════════════════════════════════════════════════════════

1. Abre Telegram (web.telegram.org o app)
2. Busca: @BotFather
3. Escribe: /newbot
4. Siguele:
   - Nombre del bot: "Casa Abuelos Bot" (o el que quieras)
   - Username: "casa_abuelos_bot" (debe ser ÚNICO y terminar en _bot)
5. Copia el TOKEN que te da (ej: 123456789:ABCDefGHIJKlmnoPQRSTuvWXYZabcdefgh)

Guardalo en un lugar seguro.

════════════════════════════════════════════════════════════════════════════════

PASO 2: CONFIGURAR TOKEN EN .ENV
════════════════════════════════════════════════════════════════════════════════

Opción A (Automatico):
  cd /home/oficina_ia/oficina_abuelos
  sed -i 's/your-telegram-bot-token-here/PEGA_TU_TOKEN_AQUI/' .env

Opción B (Manual):
  1. cd /home/oficina_ia/oficina_abuelos
  2. nano .env  (o cualquier editor)
  3. Busca: TELEGRAM_BOT_TOKEN=your-telegram-bot-token-here
  4. Reemplaza con: TELEGRAM_BOT_TOKEN=123456789:ABCDefGHIJKlmnoPQRSTuvWXYZabcdefgh
  5. Guarda (Ctrl+O, Enter, Ctrl+X en nano)

════════════════════════════════════════════════════════════════════════════════

PASO 4: USAR EL BOT EN TELEGRAM
════════════════════════════════════════════════════════════════════════════════

1. Abre Telegram
2. Busca a tu bot por @username (ej: @casa_abuelos_bot)
3. Escribe: /start
4. Haz tus preguntas

Ejemplo de preguntas:
  ✓ ¿Tienen disponible del 20 al 23 de marzo para 6 personas?
  ✓ ¿Cuánto cuesta por noche en mayo?
  ✓ ¿Aceptan perros?
  ✓ ¿Qué amenidades tiene?

TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

¿Bot no responde?
  → Verifica que el token es correcto: grep TELEGRAM_BOT_TOKEN .env
  → Revisa logs: tail -f logs/telegram_bot.log
  → Asegúrate que Ollama está corriendo

¿Errores de token?
  → Token debe empezar con números y tener :
  → Ejemplo correcto: 123456789:ABCDefGHIJKlmnoPQRSTuvWXYZabcdefgh

¿Las respuestas son muy largas?
  → Normal, Telegram divide en múltiples mensajes
  → Usa /historial para ver la conversación completa
